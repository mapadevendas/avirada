# A Virada — Live de Recrutamento, Time Paula Batista

Este repositório reúne todo o material da live **"A Virada"**, realizada em **23 de agosto**, evento gratuito de recrutamento do Time Paula Batista (Ademicon).

## Estrutura

```
├── index.html                      → Página de inscrição da live (Artifact / landing page)
├── anuncios/
│   ├── 01-problema-solucao.md      → 3 anúncios de texto, um por perfil de lead
│   ├── 02-rotina-real.md           → 2 anúncios de foto estilo orgânico, com prompts de imagem
│   └── 03-impacto-surreal.md       → 2 anúncios de foto conceitual, com prompts de imagem
├── roteiro-video.md                → Roteiro de vídeo de vendas (3 a 5 min, 6 blocos)
└── README.md                       → Este arquivo
```

## Como publicar a página

1. Abra o arquivo `index.html` em qualquer navegador para conferir.
2. Para publicar de verdade, suba este repositório em uma hospedagem simples (GitHub Pages, Vercel, Netlify) ou importe o `index.html` no Lovable para integrar com banco de dados e CRM.

## Pendências antes de rodar tráfego (marcadas no próprio HTML e nos anúncios)

- Substituir os placeholders de foto da Paula Batista (hero e bloco de autoridade)
- Substituir os 3 depoimentos de modelo por depoimentos reais de consultores
- Confirmar número de unidades, número de consultores e demais dados numéricos
- Escrever a história real da virada da Paula no Bloco 13 da página
- Trocar o link do botão de WhatsApp flutuante pelo número real (wa.me/55...)
- Trocar o link do CTA principal pelo formulário de inscrição real da live
